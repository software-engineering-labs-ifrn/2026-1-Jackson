# 💰 Meu Orçamento

**Um sistema de gerenciamento financeiro pessoal simplificado e eficiente para o controle do seu dinheiro.**

## 🌍 Contexto e Problema
Muitas pessoas enfrentam dificuldades para acompanhar suas receitas e despesas, o que frequentemente resulta em descontrole e falta de previsibilidade financeira. O **Meu Orçamento** resolve esse problema fornecendo uma plataforma acessível e intuitiva onde os usuários podem registrar, categorizar e visualizar suas transações em tempo real, facilitando o planejamento financeiro.

## ⚙️ Requisitos Gerais
Em alto nível, o sistema permite:
- Cadastro e autenticação segura de usuários.
- Registro de transações financeiras (receitas e despesas).
- Categorização das transações.
- Visualização de saldo atualizado e histórico em um painel (Dashboard).

## 👥 Papéis dos Usuários
- **Usuário (Titular da Conta):** É o ator principal e único do sistema. Ele interage com a aplicação para cadastrar sua conta, registrar suas próprias receitas e despesas, visualizar seus saldos e gerenciar seu painel financeiro. Como se trata de um sistema de finanças pessoais, o uso é estritamente individual e privado, sem acesso de terceiros ou administradores aos dados financeiros.

## 🛠️ Stack Tecnológica
O projeto foi desenvolvido utilizando as seguintes tecnologias:

**Frontend:**
- React (com Vite)
- TypeScript
- Tailwind CSS (para componentização e estilização de interfaces)

**Backend:**
- Node.js com Express
- TypeScript

**Banco de Dados:**
- MySQL / MariaDB (ou Supabase)
